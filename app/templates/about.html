About Releasarr

Releasarr is a self-hosted application designed to help you manage your music library by tracking artists and their releases. Think of it as a personal "Lidarr replacement" that allows you to:

  • Add and manage your favorite artists.
  • Fetch release information from various external sources like MusicBrainz, Discogs, and Deezer.
  • Organize and view details about each release, including tracks, year, and cover art.
  • Keep your release data updated and consistent.
  • Receive notifications about new releases or important events via configurable services like Apprise.

This project aims to provide a lightweight and customizable solution for music enthusiasts who prefer to have more control over their collection management.

You can find the source code for Releasarr on GitHub: Visit GitHub Repository

Releasarr is built using FastAPI for the backend, SQLAlchemy for database interactions, and Jinja2 for templating. It leverages external APIs to enrich your music data and Apprise for flexible notification delivery.
